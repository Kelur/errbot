api_call(method, data=None, raise_errors=True)[source]¶

Make an API call to the Slack API and return response data.

This is a thin wrapper around SlackClient.server.api_call.

Parameters:
  • method -- The API method to invoke (see https://api.slack.com/methods/).
  • raise_errors -- Whether to raise SlackAPIResponseError if the API returns an error
  • data -- A dictionary with data to pass along in the API request.
Returns:

The JSON-decoded API response
Raises:

SlackAPIResponseError if raise_errors is True and the API responds with {"ok": false}
